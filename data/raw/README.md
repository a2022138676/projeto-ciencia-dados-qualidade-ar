# Raw Data

Esta pasta contém os dados originais do projeto.

⚠ Os dados não são armazenados no GitHub devido ao limite de tamanho.

## Fonte dos Dados

Dataset: **Air Quality Dataset (UCI)**

Origem:
Kaggle → [[Kaggle → https://www.kaggle.com/datasets/uciml/air-quality-dataset](https://www.kaggle.com/datasets/rodrigopedro20/air-quality)](https://www.kaggle.com/datasets/jhonan/airqualityuci)

## Descrição

Dataset contendo medições de sensores de qualidade do ar,
incluindo concentrações de gases, temperatura e humidade.
